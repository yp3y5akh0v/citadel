# citadeldb-io

I/O layer for the [Citadel](https://github.com/yp3y5akh0v/citadel) encrypted embedded database engine. Provides the `PageIO` trait, synchronous file I/O, file locking, and optional io_uring support on Linux.

This crate is part of the Citadel workspace. Depend on the main [`citadeldb`](https://crates.io/crates/citadeldb) crate instead.
